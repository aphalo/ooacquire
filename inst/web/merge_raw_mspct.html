Merge raw spectra into a single multicolumn spectrum

Usage

merge_raw_mspct(x)

Arguments

x
raw_mspct

Description

Member spectra are sorted according to integration time stored in the inst.settings attribute aand merged into a single raw_spct object with raw counts variables named counts_1, counts_2, etc.

Note

The individual raw_spct objects contained in x must have identical values in w.length.
